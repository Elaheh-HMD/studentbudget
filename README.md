# StudentBudget

StudentBudget är en webbapplikation som hjälper studenter att hantera sin ekonomi på ett enkelt och strukturerat sätt.

## Syfte
Syftet med projektet är att skapa en enkel och pedagogisk lösning för studenter som vill få bättre kontroll över sin ekonomi.

## Målgrupp
Studenter i åldern 18–30 år som lever på begränsad inkomst.

## Funktioner (MVP)
- Registrera utgifter
- Visa en översikt av budget
- Enkel statistik

## Nästa steg
- Implementera formulär för att lägga till utgifter
- Lagra data i localStorage
- Skapa dynamiskt gränssnitt med React
